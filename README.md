# Анализ тональности отзывов на товары для выявления сильных и слабых сторон продукта

## 📋 Описание проекта
Данный проект представляет собой инструмент для анализа отзывов из CSV-файлов. Он выполняет лемматизацию текстов, определение тональности на основе рейтинга, анализ частотности слов по категориям товаров, а также предоставляет визуализацию результатов в виде графиков и облаков слов.

## Основные возможности
+ 🔍 Загрузка и фильтрация данных из CSV-файлов с автоматическим определением кодировки
+ 🌐 Фильтрация по языку (только русскоязычные отзывы)
+ 📊 Определение тональности на основе рейтинга (positive, neutral, negative)
+ 🔤 Лемматизация текста с использованием pymorphy2
+ 🛑 Управление стоп-словами (загрузка, добавление, удаление)
+ 📈 Анализ по категориям товаров с детальной статистикой
+ 📝 Сохранение отчетов в текстовом формате
+ 🖼️ Визуализация результатов (облака слов, графики, диаграммы)


## 🚀 Установка и запуск

### Требования
+ Python 3.10/3.11
+ Установленные зависимости из `requirements.txt`

### Установка зависимостей
```
pip install -r requirements.txt
```

### Запуск анализа
```
python main.py
```

### Параметры командной строки
|Параметр       |Описание                               |Значение по умолчанию |
|---------------|---------------------------------------|----------------------|
|`--input, -i `   |Путь к входному CSV файлу              |data/raw/reviews.csv  |
|`--encoding, -e` |Кодировка файла                        |None (автоопределение)|
|`--stopwords, -s`|Файл с кастомными стоп-словами         |custom_stopwords.txt  |
|`--no-progress`  |Отключить прогресс-бар                 |False                 |
|`--suggest`      |Предложить стоп-слова на основе анализа|False                 |

### Пример запуска
```
# Базовый запуск
python main.py -i data/raw/reviews.csv

# С указанием кодировки и файла стоп-слов
python main.py -i data/raw/reviews.csv -e utf-8 -s my_stopwords.txt

# С предложением стоп-слов по результатам анализа
python main.py -i data/raw/reviews.csv --suggest
```

## 🛠️ Управление стоп-словами
Стоп-слова — это слова, которые исключаются из анализа (местоимения, предлоги, общие слова). Проект поддерживает:

1. Загрузку из файла — при запуске можно указать файл со стоп-словами
2. Интерактивное управление — в процессе работы можно:

+ Просмотреть текущие стоп-слова
+ Добавить/удалить стоп-слова
+ Сохранить стоп-слова в файл
+ Загрузить стоп-слова из файла
+ Получить предложения по новым стоп-словам на основе анализа

### Формат файла стоп-слов
```
# Комментарии начинаются с #
слово1
слово2
слово3
```

Поддерживаются форматы:

+ `.txt` — по одному слову на строку
+ `.json` — список слов в формате JSON

**<summary>DiplomReviewAnalyzer (analyzer.py)</summary>**

## 📊 Аналитика по длине отзывов
Проект включает специальный анализ длины отзывов, который позволяет выявить:

+ Категории с наибольшей разницей между длиной позитивных и негативных отзывов
+ Категории, где негативные отзывы длиннее позитивных - индикатор более детального описания проблем
+ Топ-5 категорий с самыми длинными отзывами - где пользователи наиболее вовлечены

Это помогает определить, на какие категории стоит обратить особое внимание при работе с обратной связью.

## 📝 *Примечания*
1. Лемматизация выполняется с помощью pymorphy2, что позволяет приводить слова к нормальной форме
2. Стоп-слова загружаются из NLTK (русские стоп-слова) и кастомного файла
3. Тональность определяется на основе рейтинга:

+ Рейтинг 4-5 → positive
+ Рейтинг 3 → neutral
+ Рейтинг 1-2 → negative

4. Фильтрация по языку выполняется по колонке language (ищутся ключевые слова ru, рус, russian и т.д.)
